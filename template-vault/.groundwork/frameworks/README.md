# Your frameworks

This folder is your local, editable copy of the groundwork framework library. Add, remove, or rewrite any of these — they're yours.

Each framework is a markdown file with frontmatter describing when it applies and what questions it contains. To add a new one, copy any existing file, edit it, and run `groundwork-framework validate` to confirm it's well-formed.

## Seed set (v1)

- `coaching-habit-7q.md` — Michael Bungay Stanier, 7 questions
- `5-prism.md` — feelings → thoughts → needs grounding
- `grow.md` — Goal / Reality / Options / Will
- `ikigai.md` — four-circle alignment for values and offer
- `act-values.md` — Acceptance & Commitment Therapy values clarification (peak moments, life domains, direction-not-goal)

## Examples

`examples/` contains optional, project-coined frameworks shipped as inspiration, not default content. They have `.example` suffix and are not loaded automatically. Copy into this folder (drop the suffix) and adapt to make one yours.

- `examples/personal-modes.md.example` — template for a personal "modes" framework: named states you recognise in yourself mid-session (e.g. overwhelm, drift, avoidance). Replace with your own vocabulary.
